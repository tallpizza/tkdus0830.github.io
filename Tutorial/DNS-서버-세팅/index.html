    # DDNS 세팅하기

    nsupdate 명령어를 이용하는방법

    1

    $ ddns-confgen -k test.tallpizza.com

    key ’test.tallpizza.com’ { algorithm hmac-sha256; secret “<hash>” }

    위 내용을 복사후 /etc/bind 폴더 안에 test.key 파일 생성해서 내용 복사

    /etc/bind/named.conf.local 파일에

    1

    include "etc/bind/test.key"

    라인추가

    zone 파일 위치를 /etc/bind 폴더에서 /var/lib/bind 폴더로 이동해서 file location 을 변경한다

    include "/etc/bind/test.key";

zone "tallpizza.com" {
    type master;
    file "/var/lib/bind/db.tallpizza.com";
    update-policy {
      grant test.tallpizza.com wildcard *.tallpizza.com A;
      };
      #grant <keyfile name> 옵션지정
      # 위 grant 문장은 test.tallpizza.com key를 가지고 있으면 *.tallpizza.com 에 관한 A 레코드를 수정할 수 있는 권한을 주겠다 라는 뜻
    notify no;
};

    grant 옵션은 여기 를 참조하되 이해 안될지도..
