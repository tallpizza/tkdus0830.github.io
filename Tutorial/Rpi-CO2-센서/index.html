    # 라즈베리파이 조립

    (1) RPi에 LCD모니터를 연결한다(영상:HDMI, 전원:USB 사용) (2) RPi에 키보드, 마우스를 연결한다. (3) RPi에 micro SD를 밀어 넣는다. (4) RPi에 USB 아답터로 전원을 연결한다.

    # 조립

    # (1) 점퍼선 연결

    • PIN #6(GND) - 센서 GND
    • PIN #3(SDA) – 센서 SDA
    • PIN #5(SCL) - 센서 SCL
    • PIN #01(3.3V) – 센서 VDC

    # (2) I2C 작동 상태 검사

    1

    $ sudo raspi-config

    ✔️ Interfacing option에서 I2C를 Enable 한다.

    1

    $ i2cdetect -y l

    ✔️ 62 확인

    ❗ i2cdetect : command not found 일때

    1

    $ sudo apt-get install i2c-tools -y

    import time
import board
import adafruit_scd4x

i2c =board.I2C()
scd4x =adafruit_scd4x.SCD4X(i2c)

scd4x.start_periodic_measurement()
print("Waiting for first measurement....")

while True:

    if scd4x.data_ready:

        print("CO2: %d ppm"%scd4x.CO2)
		print("Temperature: %0.1f *C"%scd4x.temperature)
		print("Humidity: %0.1f %%"%scd4x.relative_humidity)
	time.sleep(1)

    • 라이브러리 지정 i2c = board.I2C() scd4x = adafruit_scd4x.SCD4x(i2c)

    • 1초마다 무한반복 while True:

      time.sleep(1)

    • 데이터 출력 print(“CO2: %d ppm”%scd4x.CO2) print(“Temperature: %0.1f *C”%scd4x.temperature) print(“Humidity: %0.1f %%"%scd4x.relative_humidity)
