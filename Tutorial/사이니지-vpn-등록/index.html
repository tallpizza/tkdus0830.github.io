    # 💡Device key 발급하기

    1. iot.anhive.net 로그인후 신규 등록 버튼을 누른다.

    2. 위 예시처럼 정보를 입력한다.

    • 운영목표는 시간을 나타낸다. (7x24는 24시간 7일)
    1. 창을 새로고침 한 후 새로운 명칭을 선택해서 들어간다.

    정보수정 -> IOT 기기 추가 버튼 -> 정보 추가.

    Device key 를 누르면 나오는 팝업창에 Location 정보를 Update한다.

    저장을 누르면 나오는 Device Key 를 복사한다. (#1)

    # 💡vpn 다운로드 링크 얻기

    192.168.10.45 으로 들어가 로그인후 → HIVE& VPN setting

    하단에 신규 란에 새로 세팅하려는 기기 이름을 입력하고 발행을 누른다.

    다운로드 링크를 복사한다. (#2)

    등록하려는 라즈베리파이의 터미널에서 $ hostname -I 를 입력하면 IP 주소가 나온다.

    얻은 IP주소를 웹브라우저 주소창에 입력한다.

    admin/1234 로 로그인후

    http://IP/setup/engineering.html 으로 접속한다.

    💡 New key

    New key 입력란에 (#1) 에서 복사한 Device key 를 붙여넣기 한다.

    💡 VPN cert resource

    VPN cert resource에 (#2)에서 얻은 vpn 다운로드 링크를 붙여넣기 한다.

    Update 를 누르면 알림창이 뜨고 잠시후 사이니지가 재부팅을 한다.

    192.168.10.45 → HIVE& VPN setting 에서 새로 목록이 생성되고, 상태를 확인 할 수 있다.

    사이니지 설치는 192.168.10.63 wiki → 참조 SurpirseBOX의 인터넷액자 설치 스크립트(PiXEL) 참조
